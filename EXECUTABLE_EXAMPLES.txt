CLEARCOM EXECUTABLE EXAMPLES (ADVANCED)
========================================

Run any file:
  ./.venv/bin/python main.py <file.mc>
Or in web UI:
  ./.venv/bin/python web_app.py

----------------------------------------
1) BASIC DECLARATION + ASSIGNMENT
----------------------------------------
int a;
float b;
a = 5;
b = 2.5;
print("a=", a);
print("b=", b);

----------------------------------------
2) INITIALIZED DECLARATIONS
----------------------------------------
int x = 10;
float y = 3.5;
print("x=", x);
printf("y=%f", y);

----------------------------------------
3) ARITHMETIC EXPRESSIONS
----------------------------------------
int a = 10;
int b = 3;
int c = a + b * 2;
print("c=", c);
printf("mod=%d", a % b);

----------------------------------------
4) CONSTANT FOLDING DEMO
----------------------------------------
int folded;
folded = 2 + 3 * 4;
print("folded=", folded);

----------------------------------------
5) IF / ELSE
----------------------------------------
int a = 5;
if (a > 3) {
    print("branch=", 1);
} else {
    print("branch=", 0);
}

----------------------------------------
6) WHILE LOOP
----------------------------------------
int i = 0;
while (i < 5) {
    print("i=", i);
    i = i + 1;
}

----------------------------------------
7) FOR LOOP
----------------------------------------
int sum = 0;
int i;
for (i = 0; i < 5; i = i + 1) {
    sum = sum + i;
}
printf("sum=%d", sum);

----------------------------------------
8) NESTED CONTROL FLOW
----------------------------------------
int i;
for (i = 1; i <= 5; i = i + 1) {
    if (i % 2 == 0) {
        print("even=", i);
    } else {
        print("odd=", i);
    }
}

----------------------------------------
9) PRINTF + PRINT VARIANTS
----------------------------------------
int v = 42;
printf("value=%d", v);
print("value=", v);
print("value_addr_style=", &v);
print("done");

----------------------------------------
10) COMMENTS SUPPORTED
----------------------------------------
// single-line comment
int a = 1;
/* block
   comment */
a = a + 9;
print("a=", a);

----------------------------------------
11) EXECUTION TRACE DEMO (Web UI)
----------------------------------------
Use this code and enable Trace checkbox in UI:
int n = 0;
while (n < 3) {
    print("n=", n);
    n = n + 1;
}

----------------------------------------
12) ERROR DETECTION DEMO (KEYWORD TYPO)
----------------------------------------
innt a;        // typo warning suggestion: int
int b;
b = 5;         // if b is never read later, unused warning can appear

Expected: compile error "Unknown type name 'innt'" plus typo suggestion.

----------------------------------------
12B) WARNING-ONLY DEMO (UNUSED VARIABLE)
----------------------------------------
int a;
int b;
b = 5;
print("b=", b);

Expected: successful compile with warning for unused variable 'a'.

----------------------------------------
13) RUNTIME ERROR HANDLING DEMO
----------------------------------------
int x = 10;
int y = 0;
int z;
z = x / y;
print("z=", z);

Expected: runtime error for division by zero.

----------------------------------------
14) FORMATTER DEMO (Web UI)
----------------------------------------
Paste ugly code then click Format Code:
int a=5;int b=2; if(a>b){print("ok",a);}else{print("no",b);} 

----------------------------------------
15) FULL PROGRAM STYLE WITH main
----------------------------------------
#include<stdio.h>
int main(){
    int a = 2;
    int b = 3;
    int c = a + b;
    printf("c=%d", c);
}
